# Traffic Jam

An HTML/JavaScript animation I made for my bus-obsessed nephew.

[View it &raquo;](http://)

## Run it locally

Set up:

```
$ git clone http://github.com/mdb/traffic_jam
$ cd traffic_jam
$ bower install
$ bundle install
$ middleman
```

View it in your web browser at [http://localhost:4567](http://localhost:4567).
